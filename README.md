# Приложение Strava
____
# Описание:
Клиент для сервиса Strava, с помощью которого можно получать информацию о зарегистрированном пользователе, а также посмотреть его тренировки.
____
# Используемые технологии:
--**ViewModel**

--**RecyclerView**

--**Glide**

--**OAUTH2.0**

--**Coroutines**

--**Retrofit**

--**OkHTTP**

--**RoomDao**

--**Navigation Component**
